edible-combobox

A searchable dropdown that replaces <select> when the list needs filtering. Type to narrow options; navigate with arrow keys; confirm with Enter.

Basic

Declare <option> children — they are consumed on connect.

Pre-selected value

Set .value via JavaScript after the element upgrades.

Disabled

Add the disabled attribute to prevent any interaction.

Inside a form

The component participates in form submission via a hidden <input>. The selected value is included in FormData under the name key.
